X-GIS Language

지도를 위한 프로그래밍 언어 — 3가지 사용 방법

1. 인라인 코드

<xgis-map>
  let world = load("countries.geojson")
  show world { fill: #3a6b4e, stroke: #2a4a3a, 1px }
</xgis-map>
let world = load("data/countries.geojson") show world { fill: #3a6b4e stroke: #2a4a3a, 1px }

2. .xgis 파일 참조

<xgis-map src="hello.xgis"></xgis-map>

3. script 태그 참조

<script type="text/xgis" id="ocean-map">
  let world = load("data/countries.geojson")
  show world { fill: #2c5f7c, stroke: #1a3d52, 1px }
</script>
<xgis-map script="ocean-map"></xgis-map>

4. 컴파일된 바이너리 (.xgb)

$ xgisc compile hello.xgis -o hello.xgb   (126B → 62B, 51% smaller)
<xgis-map src="hello.xgb"></xgis-map>

5. Projections (클라이언트 API)

같은 X-GIS 코드, 다른 프로젝션 — JS에서 런타임 전환

const map = new XGISMap(canvas)
await map.run(xgisCode)
map.setProjection('equirectangular')  // 런타임 전환!

프로젝션은 항상 화면 중심 기준. 패닝하면 프로젝션 중심이 따라감.
